# Choose a language to start a new project

## 1. Technical fit

Cân nhắc các phương án xem nó có giải quyết được vấn đề hay không.
Ví dụ làm data rất phù hợp với Python, làm windows thì dùng C#, làm high traffic system thì dùng go, rust, làm frontend thì dùng node.

## 2. Resource & Maintainable

Có nhân sự để làm phần này không? 
Làm xong rồi có dễ maintain hay không?

## 3. Time to market

Cân nhắc thời gian dev có lâu không?
Có template nào support gần hết, mình chỉ vào edit một chút không?

## 4. Operation Cost

Infra cost, cân nhắc cloud hay local, free or charging
