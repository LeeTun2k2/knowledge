## Protobuf

Is a method to serialize data like JSON, XAML but simpler.
Data structure: is defined in a schema in .proto
Serialization Format: Binary
Language Support: Multi-Language.
Backward & Forward Compatibility: Safe to add new fields -> easily scale

## Usage

When require a high performance serialization/deserialization.
